# CORONA - CHART

## 철지난 코로나19 데이터를 다루는 이유

 - [x] 차트를 한번 다뤄보고 싶었다.
 - [x] 정보 공유가 가능한 웹사이트였으면 좋겠다고 생각했다.
 - [x] 일본 여행 계획이 있는데, 참고할 겸 일본 코로나 데이터를 이용해 그래프로 참고할 만한 페이지가 있었으면 좋겠다고 생각했다.
